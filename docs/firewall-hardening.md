# Firewall Hardening Guide

## 1. Change Default Credentials

Use strong passwords.

## 2. Restrict Admin Access

Allow login only from trusted IPs.

## 3. Disable Telnet / HTTP

Use SSH and HTTPS.

## 4. Enable Logging

Monitor denied traffic and admin changes.

## 5. Update Firmware

Keep firewall updated.

## 6. Remove Unused Rules

Delete ANY-ANY rules.

## 7. Backup Configurations

Take regular backups.
